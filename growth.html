Growth

This work was very challenging for me. To make my game, I needed to program and reprogram the movements of each element to time perfectly. It was hard to balance the game so that it is playable but challenging. I grew from this work because I learned how to customize my game idea and make it interesting and fun. I learned how to use the collide block, which was a challenge to get right. One obstacle I got past was the placement of the platforms, because they led to the wizard getting stuck in the top of the screen. I recoded the platforms repeatedly to find a good rhythm. I put in a lot of effort, continually tweaking it, which led to its success as a game. If I were to improve this project now, I would further adjust the platforms and wizard’s velocity to make the game a little more challenging.
